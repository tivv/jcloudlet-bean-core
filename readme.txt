JCloudlet Bean Core Module

Version 1.0-SNAPSHOT

This module is a part of JCloudlet Web Framework, but it can be used separately as well.

Usage

In order to include this module add following dependency to your pom.xml:

    <dependencies>
        <dependency>
            <groupId>org.jcloudlet</groupId>
            <artifactId>jcloudlet-bean-core</artifactId>
            <version>1.0-SNAPSHOT</version>
        </dependency>
    </dependencies>

Also, do not forget to set maven-compiler plugin settings to use Java version 1.6+.
